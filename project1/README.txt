to run the project
- write in terminal:
	. npm build
	. npm run start
- then open in your browser "http://localhost:3000/api"
- to add image by width and hight and the name of image seprated by ","
	ex:http://localhost:3000/data?data=500,500,flower

- you can delete the thumbnail folder and the project create it and add in it the new images.
- if the image existe not create it again, if not , the project create it and when reload show it.

to run the test
- npm run build
- npm run jasmine
- npm run test
